## IBM University Engagement Project

**Exploring the Power of Agentic AI with IBM Cloud and Orchestrate**

---

# 📌 Project Overview

Tejas Nutrition Agent is an AI-powered virtual nutrition assistant developed using **IBM watsonx Orchestrate** and **IBM Granite Foundation Models**.

The project helps users receive personalized nutrition guidance, healthy meal recommendations, calorie estimates, and fitness diet suggestions through a conversational AI interface.

The goal is to encourage healthier lifestyles by providing reliable, personalized nutrition advice in real time.

---

# 🎯 Problem Statement

Many people struggle to maintain healthy eating habits because nutrition information is scattered across multiple sources and often lacks personalization.

Users commonly face challenges such as:

- Planning balanced diets
- Understanding calorie requirements
- Gym nutrition planning
- Meal recommendations
- Healthy lifestyle management

The Nutrition Agent solves these challenges by acting as an intelligent AI-powered nutrition coach.

---

# 🚀 Solution

The project uses IBM watsonx Orchestrate to create an AI agent capable of understanding natural language questions and generating personalized nutrition recommendations.

The system performs the following tasks:

- Personalized diet planning
- Healthy meal suggestions
- Coffee optimization advice
- Gym nutrition guidance
- Macronutrient recommendations
- Calorie estimation
- Healthy lifestyle coaching
